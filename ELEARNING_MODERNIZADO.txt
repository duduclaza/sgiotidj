## 🚀 Como Começar

### Passo 1: Ver o Preview
Abra em seu navegador: **`/elearning-design-preview.html`**

### Passo 2: Incluir CSS em cada view
```php
<link rel="stylesheet" href="/assets/elearning-modern.css?v=<?= time() ?>">
```

### Passo 3: Usar os componentes
```html
<!-- Exemplo 1: Button -->
<button class="el-btn el-btn-primary">Ação</button>

<!-- Exemplo 2: Card com conteúdo -->
<div class="el-card">
    <div class="el-card-header">
        <h3 class="el-card-title">Título</h3>
    </div>
    <div class="el-card-body">Conteúdo...</div>
</div>

<!-- Exemplo 3: Course Card completo -->
<div class="el-course-card">
    <div class="el-course-image">📖</div>
    <div class="el-course-content">
        <h3 class="el-course-title">Curso</h3>
        <div class="el-progress-container">
            <div class="el-progress-bar" style="width: 75%;"></div>
        </div>
    </div>
</div>
```

### Passo 4: Atualizar próximas views
Use os arquivos como template (dashboard + meus_cursos já estão prontos)

## 💡 Tips & Tricks

### Personalizar cores variáveis CSS
```css
:root {
  --primary-500: #0ea5e9;     /* Change to your color */
  --secondary-500: #7c3aed;   /* Change to your color */
}
```

### Adicionar nova variação de badge
```css
.el-badge-custom {
  background: var(--custom-100);
  color: var(--custom-700);
}
```

### Fazer card ativo/selecionado
```html
<div class="el-card el-card-active">Conteúdo</div>
```

### Customizar tamanhos de fonte
Use as 8 variações de texto ou estenda com Tailwind:
```html
<h1 class="el-h1">Heading 1</h1>
<p class="text-2xl font-bold">Custom Heading</p>
```
